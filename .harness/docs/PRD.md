# PRD: bid-system

---

## 목표

입찰업체 담당자가 나라장터 입찰에서 엑셀 루틴처럼 수동으로 하던 입단가 산정을 AI(Monte Carlo 시뮬레이션)가 대신하여 낙찰 확률을 높인다.

---

## 사용자

| 페르소나 | 설명 | 주요 니즈 |
|---------|------|----------|
| 입찰 담당자 | 나라장터 입찰 공고를 검색·신청하는 업무 담당자 | 빠른 입단가 추천, 경쟁사 현황 파악, 낙찰 이력 관리 |
| 관리자 | 시스템 사용자 및 데이터 관리 권한 보유자 | 사용자 관리, 공고 데이터 동기화 상태 확인 |

---

## 핵심 기능

1. **입단가 추천** — Monte Carlo 시뮬레이션(30,000회)으로 복수예가 15→4개 추첨 메커니즘을 모사, 낙찰 확률과 4가지 전략(공격형·보수형·중앙집중형·랜덤형)을 제시한다.
2. **입찰 공고 목록 조회** — 나라장터 OpenAPI에서 수집한 공고를 업종·지역·금액·날짜로 필터링하여 조회한다.
3. **낙찰 이력 관리 (My입찰)** — 내가 신청한 입찰의 낙찰 여부와 이력을 추적한다.
4. **경쟁사 분석** — 동일 공고에 참여한 경쟁업체의 입찰 패턴(낙찰률·평균 낙찰가율)을 시각화한다.
5. **대시보드** — 낙찰률 추이, 금액대별 성공률 등 핵심 지표를 한눈에 확인한다.
