## 개요

데이터 내보내기 기능 3종 추가 — 투찰 이력 Excel, 최종 추천 레포트 PDF, 통계 CSV.

---

## 변경 사항

### Step 1 — 투찰 이력 Excel 내보내기

**파일:** `bid-system/backend/app/api/v1/my_bids.py`  
**파일:** `bid-system/frontend/src/pages/MyBidsPage.tsx`  
**파일:** `bid-system/frontend/src/api/index.ts`

- `GET /my-bids/export/excel` 엔드포인트 신규 (openpyxl + StreamingResponse)
- 11개 컬럼: 공고번호·공고제목·발주처·입찰일·기초금액·제출투찰률·추천투찰률·결과·실제낙찰률·격차·비고
- 파란색 헤더(1E40AF), 흰색 볼드 폰트
- `ExcelDownloadButton` 컴포넌트 추가 (MyBidsPage 헤더 우측)
- `myBidsApi.exportExcel()` → Blob → `URL.createObjectURL` 패턴

### Step 2 — 최종 추천 레포트 PDF 출력

**파일:** `bid-system/frontend/src/pages/TenderRecommendPage.tsx`

- jsPDF + html2canvas 동적 임포트 (코드 스플리팅 유지)
- `printRef` 래퍼로 전체 카드 영역 캡처
- 멀티 페이지 A4 자동 처리 (페이지 오프셋 루프)
- `PDF 출력` 버튼 헤더 우측 추가, 출력 중 Loader2 스피너
- 파일명: `투찰추천_{bid_id}.pdf`

### Step 3 — 통계 CSV 내보내기

**파일:** `bid-system/frontend/src/pages/StatisticsPage.tsx`

- `downloadCsv(filename, rows)` 유틸 함수 추가 (BOM UTF-8, Excel 한글 호환)
- `handleCsvDownload()`: 현재 탭 기준 데이터 내보내기
  - 개요 탭 → `발주처통계_{N}개월.csv` (발주처·입찰건수·평균낙찰률·평균경쟁사수)
  - Level 1 탭 → `공종별통계_{N}개월.csv` (공종명·입찰건수·평균낙찰률·평균경쟁사수·총금액)
- `CSV` 버튼 기간 버튼 오른쪽에 추가 (개요·Level 1 탭에서만 표시)
- 신규 의존성 없음 (순수 클라이언트 사이드 문자열 처리)
